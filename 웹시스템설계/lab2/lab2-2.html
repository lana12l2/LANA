html

하이퍼 텍스트 마크업 언어HTML( HyperText Markup Language )는 웹 페이지의 구조를 정의하는 마크업 언어이다. 웹사이트를 만들 때, 텍스트, 이미지, 동영상, 링크, 버튼 등을 배치하고 정리하는 역할이다. HTML 자체는 디자인이나 동적인 기능을 담당하지 않고, 단순히 웹 페이지의 뼈대를 만든다. HTML은 기본적으로 웹 브라우저가 이해할 수 있는 코드로 구성된다. 브라우저(Chrome, Edge, Firefox 등)는 HTML 코드를 읽고 해석해서 우리 눈에 보이는 웹 페이지로 변환한다. 예를 들어, HTML을 사용해서 제목, 문단, 이미지, 목록, 표, 버튼 등을 배치하고, CSS를 추가하면 스타일을 지정할 수 있다. HTML은 단순한 텍스트가 아니라 **태그(tag)**를 사용해서 문서의 구조를 표현한다. 각 태그는 특정한 역할을 가지고 있고, 브라우저는 이 태그를 해석해서 화면에 표시한다.

css

캐스캐이딩 스타일 시트 css ( Cascading Style Sheets )는는 웹 페이지의 디자인과 레이아웃을 담당하는 스타일 시트 언어이다. HTML이 웹 페이지의 구조를 정의한다면, CSS는 색상, 글꼴, 크기, 간격, 배경, 애니메이션 등 웹 요소의 스타일을 지정하는 역할을 한다. CSS의 역할 CSS는 HTML 요소의 모양과 배치를 조정하는 기능을 한다. 이를 통해 웹 페이지를 보기 좋고 사용자 친화적으로 디자인할 수 있다. CSS는 폰트 스타일, 색상, 테두리, 여백, 정렬, 배경 이미지 등의 다양한 속성을 조절할 수 있으며, 반응형 디자인을 적용하여 다양한 화면 크기(PC, 태블릿, 모바일)에 맞춰 웹 페이지를 최적화할 수도 있다.

javascript

자바스크립트(JavaScript)는 웹 페이지를 동적으로 만들기 위한 프로그래밍 언어이다. HTML이 웹 페이지의 구조를 담당하고, CSS가 디자인을 담당한다면, 자바스크립트는 웹 페이지에 동작(인터랙션)을 추가하는 역할을 한다. 브라우저에서 실행되는 대표적인 스크립트 언어이며, 현재는 웹뿐만 아니라 서버(Node.js), 모바일, 게임, AI, IoT 등 다양한 분야에서 활용된다.
